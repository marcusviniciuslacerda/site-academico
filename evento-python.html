Minicurso de Python

O Minicurso de Python tem como objetivo introduzir os participantes aos conceitos fundamentais da programação utilizando uma das linguagens mais populares do mercado. Durante o evento serão abordados lógica de programação, variáveis, estruturas de decisão, repetição e criação de pequenos projetos práticos.

Data: 12 de Junho de 2026

Horário: 08h às 12h

Local: Laboratório de Informática 02

Carga Horária: 4 horas

Vagas: 40 participantes

Programação

Horário Atividade
08:00 Introdução ao Python
09:00 Variáveis e Operadores
10:00 Estruturas de Decisão
11:00 Estruturas de Repetição e Exercícios

Instrutor

Prof. Carlos Mendes

Professor de Programação e Desenvolvimento de Sistemas.
