# Site Acadêmico

## 📚 Sobre o projeto

Este projeto foi desenvolvido como trabalho final da disciplina de Front-End I do Instituto Federal do Piauí (IFPI).

O objetivo foi criar um site acadêmico responsivo utilizando HTML5 e CSS3, aplicando os principais conceitos estudados durante a disciplina, como HTML semântico, Flexbox, Grid Layout, formulários e versionamento com GitHub.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Git
- GitHub
- GitHub Pages

---

## 📄 Funcionalidades

- Página inicial
- Menu de navegação
- Cabeçalho e rodapé padronizados
- Páginas interligadas
- Layout responsivo
- Galeria organizada com Grid Layout
- Alinhamento utilizando Flexbox
- Formulário com validação HTML5
